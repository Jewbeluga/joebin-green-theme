# Joebin Green

A custom VS Code dark theme with a green-focused palette, designed for comfortable long coding sessions.

## Features

- Dark theme optimized for readability
- Green-accented syntax highlighting
- Clean contrast for LaTeX and general coding

## Installation

### From VSIX (local)
1. Open VS Code
2. Go to Extensions
3. Click `...` → **Install from VSIX**
4. Select the `.vsix` file

### From source
Clone the repository and package with `vsce`:
```bash
npx @vscode/vsce package
